# Proyecto 3D_matt

## Descripción
Este proyecto consiste en el desarrollo de un sitio web para el emprendimiento 3D_matt, dedicado a la venta de productos realizados mediante impresión 3D.

El sitio permite mostrar productos, servicios y formas de contacto con el cliente.

## Objetivo
El objetivo es brindar una presencia digital al emprendimiento, permitiendo mejorar su alcance y organización.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Estructura del proyecto
El proyecto cuenta con múltiples páginas:
- index.html (inicio)
- catalogo.html
- servicios.html
- proyectos.html
- contacto.html

## Cómo ejecutar el proyecto
1. Descargar el repositorio
2. Abrir el archivo index.html en un navegador
